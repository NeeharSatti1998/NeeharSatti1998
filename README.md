### 🚀 Highlighted Projects

#### 🧾 [Credit Score Prediction](https://github.com/NeeharSatti1998/credit_score_prediction)
Automated ML pipeline to classify creditworthiness using Python, sklearn, and model evaluation metrics.

#### 💬 [RAG Book Chatbot](https://github.com/NeeharSatti1998/rag-book-chatbot)
Retrieval-Augmented Generation chatbot built with ChromaDB + Mistral LLM for intelligent book-based Q&A.

#### 💹 [Apple Stock Sentiment Prediction](https://github.com/NeeharSatti1998/stock-sentiment-prediction)
Predicted Apple stock movements using FinBERT and VADER sentiment models. Deployed with FastAPI and Streamlit on AWS.

#### 🔍 [Customer Churn Prediction](https://github.com/NeeharSatti1998/churn-prediction)
Boosted churn prediction accuracy using XGBoost and Random Forest. Integrated REST API and Power BI dashboards.
